## 项目简介
这是一个基于 C 语言实现的 `key=value` 格式字符串解析项目，核心功能封装在 `parser/` 目录下，提供完整的解析状态判断、键值对存储和内存释放功能，可被主程序 `main.c` 直接调用，适用于简单配置项解析、数据格式转换等场景。

## 环境要求
- 编译器：GCC（Linux）
- 版本控制：Git
- 运行环境：Linux 终端
